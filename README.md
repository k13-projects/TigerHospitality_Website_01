# TIGER HOSPITALITY Website

A modern, responsive website for TIGER HOSPITALITY featuring premium animations, interactive elements, and accessibility features.

## Features

- **Animated Loading Screen** - TIGER HOSPITALITY logo reveal animation
- **Mobile Hamburger Menu** - Smooth slide-in navigation for mobile devices
- **Interactive Contact Form** - Full validation with success/error messaging
- **Custom Cursor** - Gold ring cursor with hover effects
- **Google Maps Integration** - Restaurant location mapping (API key required)
- **Flip Card Concepts** - Interactive restaurant showcase cards
- **Scroll Animations** - Hero shrink effect, collage fade-in, stat counters
- **Accessibility Features** - ARIA labels, keyboard navigation, skip links, reduced motion support
- **Performance Optimized** - Lazy loading ready, optimized images

## Technologies

- HTML5
- CSS3 (with custom animations)
- Vanilla JavaScript
- Google Maps API (optional)
- Google Fonts (Bebas Neue, Montserrat, Playfair Display)

## Setup

1. Clone the repository
2. Open `index.html` in a browser or serve with a local server
3. (Optional) Add Google Maps API key in line 1355 to enable interactive maps

## Local Development

Run a local server:
```bash
python -m http.server 3000
```

Then visit: `http://localhost:3000`
